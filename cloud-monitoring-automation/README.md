# 📊 Cloud Monitoring & Automation

## 📌 Project Overview

This project demonstrates monitoring and automation of AWS resources using Amazon CloudWatch and Amazon SNS. It focuses on monitoring EC2 instances, configuring alarms, and sending automated notifications.

---

## 🛠️ AWS Services Used

- Amazon CloudWatch
- Amazon EC2
- Amazon SNS
- CloudWatch Alarms
