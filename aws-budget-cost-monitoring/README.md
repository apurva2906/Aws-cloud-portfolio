# 💰 AWS Budget & Cost Monitoring

## 📌 Project Overview

This project demonstrates AWS cost management by configuring AWS Budgets and billing alerts. It helps monitor AWS spending, receive notifications when budget thresholds are exceeded, and understand service-wise costs.

---

## 🛠️ AWS Services Used

- AWS Budgets
- Amazon SNS
- AWS Billing Dashboard
