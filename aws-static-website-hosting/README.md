# 🌐 AWS Static Website Hosting

## 📌 Project Overview

This project demonstrates how to host a static website using Amazon S3 and Amazon CloudFront. The website is configured for secure, scalable, and highly available content delivery.

---

## 🛠️ AWS Services Used

- Amazon CloudFront
