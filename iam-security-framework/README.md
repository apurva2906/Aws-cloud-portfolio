# 🔐 IAM Security Framework

## 📌 Project Overview

This project focuses on implementing a secure Identity and Access Management (IAM) framework in AWS. It demonstrates user authentication, authorization, role-based access control, and security best practices using AWS IAM.

---

## 🛠️ AWS Services Used

- AWS IAM
- IAM Users
- IAM Groups
- IAM Roles
- IAM Policies
- Multi-Factor Authentication (MFA)
