
# ☁️ AWS Infrastructure Deployment

## 📌 Project Overview

This project demonstrates the deployment of a secure and scalable AWS infrastructure using core AWS services. The infrastructure was built using the AWS Management Console and follows cloud best practices for networking, security, monitoring, and storage.

---

## 🛠️ AWS Services Used

- Amazon EC2
- Amazon VPC
- Public & Private Subnets
- Internet Gateway
- Route Tables
- Security Groups
- Amazon RDS
